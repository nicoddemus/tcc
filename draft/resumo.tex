\chapter{Resumo}

A linguagem de programação C++, apesar de ser auto-nível e possibilitar a produção de um código de 
execução bastante eficiente, impõe ao desenvolvedor um custo relativamente alto: o tempo necessário
para a compilação do seu código. Técnicas agora disponíveis, especialmente \textit{Template Metaprogramming},
enquanto promovem a geração de código genérico, contribuem ainda mais para este custo.

Este trabalho tem como o objetivo o estudo e a criação de uma ferramenta que
distribua o processo de compilação entre várias máquinas conectadas através de uma rede
local, diminuindo assim o tempo e os custos necessários para o mesmo.
